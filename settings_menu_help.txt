~~ Options in Settings ~~
Set Start: set the starting location for the pathfinder
Set End: set the destination for the pathfinder
Preference: choose whether the best route is determined by being the cheapest, fastest or having the fewest segments
Set Avoid Modes: Set which modes of transport to avoid
Clear Preferences: Clear start, end and avoid modes
Print Settings: Print the above settings
Help: Print this page
Return: Returns to the main menu

Important Note: Settings is not available until you Upload Map on the main menu
